# self-decoder
<h3>written by python</h3><hr>
for execute in windows run <code>py decode</code> and <code>py encode</code>.</br> you can give your text as arg like <code>py encoder hello world</code> or <code>py decode 7342624183218430834372417211740394008200821264416343524163117221</code>.</br>or you can enter your text as input after runing script.<hr>
for execute in linux run <code>chmod +x decode</code> and <code>chmod +x encode</code>.</br>then run <code>./encode</code> or <code>./decode</code><hr>
you can install it and use it on terminal in every path using</br><code>sudo cp encode /usr/local/bin</code> and <code>sudo cp decode /usr/local/bin</code></br>and use it as <code>encode</code> and <code>decode</code>
